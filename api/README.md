# ตั้งค่าฐานข้อมูลบน Plesk (ทำครั้งเดียว)

โฟลเดอร์ `api/` คือ backend เล็กๆ (PHP + MySQL) ที่ทำให้ข้อมูลถูกเก็บ **บนเซิร์ฟเวอร์** —
ทุกคนที่เปิดเว็บจะเห็นข้อมูลชุดเดียวกัน (เดิมเก็บแยกในเบราว์เซอร์ของแต่ละคน)

> ทำตามนี้ทีละขั้น ไม่ต้องรู้ SQL — ระบบสร้างตารางให้เองอัตโนมัติ

## 1) สร้างฐานข้อมูลใน Plesk

1. เข้า Plesk → เลือกโดเมนของเว็บนี้ → เมนู **Databases** → **Add Database**
2. ตั้งชื่อฐานข้อมูล เช่น `ads_optimizer`
3. สร้าง **Database user** + รหัสผ่าน (จดไว้)
4. กด OK — จะได้ค่า 4 อย่าง: host (ปกติ `localhost`), ชื่อ DB, user, password

## 2) ใส่ค่าฐานข้อมูลลงไฟล์ config

1. ไปที่ **File Manager** ของโดเมน → เข้าโฟลเดอร์ `api/`
2. **คัดลอก** ไฟล์ `config.sample.php` แล้วเปลี่ยนชื่อสำเนาเป็น `config.php`
3. เปิด `config.php` แก้ค่า 4 บรรทัดฐานข้อมูล (`host`, `name`, `user`, `pass`) ให้ตรงกับที่ได้จากขั้นที่ 1 → Save

> ⚠️ ไฟล์ `config.php` มีรหัสผ่านฐานข้อมูล จึงถูกตั้งให้ **ไม่ขึ้น git** (อยู่ใน `.gitignore`)
> ต้องสร้างไฟล์นี้บนเซิร์ฟเวอร์เองเท่านั้น — ไม่ต้อง (และห้าม) push ขึ้น GitHub

## 3) ตรวจว่าใช้งานได้

เปิด URL นี้บนเบราว์เซอร์ (แทน `your-domain.com` ด้วยโดเมนจริง):

```
https://your-domain.com/api/data.php
```

- ถ้าเห็นข้อความ JSON เช่น `{"ok":true,"store":{},"targets":{}}` = **ใช้ได้แล้ว** ✅
  (ตารางในฐานข้อมูลถูกสร้างให้อัตโนมัติตอนเรียกครั้งแรก)
- ถ้าเห็น `"error":"เชื่อมต่อฐานข้อมูลไม่สำเร็จ..."` = ค่าใน `config.php` ยังไม่ถูก ลองตรวจขั้นที่ 2 ใหม่

เสร็จแล้ว! เปิดหน้าเว็บหลักได้เลย ข้อมูลที่อัปโหลด/ลบ จะถูกบันทึกบนเซิร์ฟเวอร์และทุกคนเห็นเหมือนกัน

---

## หมายเหตุสำคัญ

- **เปิดให้ทุกคนเข้าใช้งานได้ ไม่มีหน้า login** — ใครมี URL ของเว็บก็ดู/แก้/ลบข้อมูลได้
  ถ้าภายหลังอยากจำกัดการเข้าถึง บอกได้ จะเพิ่มระบบ login กลับให้
- **รันในเครื่อง (local) ด้วย `python -m http.server` ได้เหมือนเดิม** — เพราะไม่มี PHP ทำงาน
  เว็บจะ fallback ไปเก็บใน localStorage ของเครื่องนั้นโดยอัตโนมัติ (ไม่พัง, แค่ไม่ใช่ข้อมูลรวมศูนย์)
- การต่อ Plesk Git auto-deploy: เวลามี push ใหม่ Plesk จะดึงโค้ดมาแทนที่ — แต่ `config.php`
  ที่สร้างไว้บนเซิร์ฟเวอร์จะ **ไม่ถูกแตะ** (เพราะไม่ได้อยู่ใน git) จึงตั้งค่าแค่ครั้งเดียวพอ
